# Neovim Configuration

Personal neovim configuration, written in Lua.

## Plugins

### Git integration

#### vim-gitgutter

> [Plugin reference](https://github.com/airblade/vim-gitgutter)

| keys         | mapping                                |
| ------------ | -------------------------------------- |
| [c           | Previous hunk                          |
| ]c           | Next hunk                              |
| \<leader>hp  | Preview hunk                           |
| \<leader>hs  | Stage hunk                             |
| \<leader>hu  | Undo hunk                              |
